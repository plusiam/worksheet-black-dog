# 검은 강아지 학습지 🐾

그림책 **'검은 강아지'**를 활용한 생명존중 수업 학습지입니다.

## 개요

- **대상**: 초등학교 3~4학년
- **주제**: 생명은 소중해요
- **형식**: A4 양면 인쇄용 HTML

## 학습 흐름

1. **나의 마음 들여다보기** - 강아지를 키우고 싶은 이유 탐색
2. **강아지는 어떤 존재일까?** - 돌봄이 필요한 이유 이해
3. **유기된 강아지의 마음 상상하기** - 감정 공감 및 그림 표현
4. **나의 마음과 강아지의 필요 연결하기** - 관점 비교
5. **생명을 존중하는 나의 약속** - 실천 다짐

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. `Ctrl + P` (또는 `Cmd + P`)로 인쇄
3. 용지: A4, 양면 인쇄
4. 여백: 없음 또는 최소
5. 배경 그래픽 포함 체크
